## 二、功能

项目中已实现的功能如下：

+ 语法词法分析输出AST
+ 语义分析
    + 符号重复定义
    + 循环继承
    + 类型未定义
    + 符号未定义

+ 错误处理
    + 输出错误类型
    + 错误恢复

+ 简易的后端
